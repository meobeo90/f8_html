Hướng dẫn xây dựng ứng dụng Todo List với HTML, CSS, và JavaScript

Bước 1: Tạo khung HTML cơ bản

Trước tiên, chúng ta cần tạo cấu trúc HTML để hiển thị các phần tử của ứng dụng như ô nhập, nút thêm và danh sách công việc. 

    <!DOCTYPE html>
    <html lang="vi">
    <head>
    <meta charset="UTF-8">
    <title>Todo List</title>
    <link rel="stylesheet" href="styles.css">
    </head>
    <body>
    <div class="todo-container">
        <h1>Todo List</h1>
        <input type="text" id="todo-input" placeholder="Nhập công việc...">
        <button id="add-btn">Thêm</button>
        <ul id="todo-list"><ul>
    </div>
    <script src="script.js"><script>
    </body>
    </html>

Giải thích: HTML trên tạo ra giao diện cơ bản gồm ô nhập liệu, nút thêm và danh sách <ul> nơi các công việc sẽ hiển thị.

Bước 3: Thêm chức năng bằng JavaScript

Chúng ta sẽ thêm logic để thêm và xoá công việc bằng file script.js. 


    const input = document.getElementById("todo-input");
    const addBtn = document.getElementById("add-btn");
    const list = document.getElementById("todo-list");

    addBtn.addEventListener("click", function () {
      const task = input.value.trim();
      if (task !== "") {
        const li = document.createElement("li");
        li.textContent = task;

        const deleteBtn = document.createElement("button");
        deleteBtn.textContent = "×";
        deleteBtn.onclick = () => li.remove();

        li.appendChild(deleteBtn);
        list.appendChild(li);
        input.value = "";
      }
    });

Giải thích: Khi nhấn nút "Thêm", JavaScript sẽ tạo một mục mới và thêm vào danh sách. Mỗi mục có nút "×" để xoá dòng tương ứng.

Tổng kết

Chúng ta đã hoàn thành một ứng dụng Todo List đơn giản với HTML, CSS và JavaScript. Bạn có thể nâng cấp thêm như lưu vào localStorage, đánh dấu hoàn thành, sắp xếp công việc,... để hoàn thiện hơn nữa.
